Bem-vindo ao repositório do meu portfólio pessoal! 👋

Este projeto foi construído para ser a minha vitrine profissional, onde compartilho um pouco sobre quem eu sou, as tecnologias que domino e, principalmente, os projetos que desenvolvi ao longo da minha carreira na engenharia de software.

## 🎯 Objetivo

Criar uma experiência fluida, rápida e responsiva para apresentar meus projetos, currículo e artigos de forma profissional, demonstrando na prática as minhas habilidades de desenvolvimento front-end/full-stack.

## 🛠️ Tecnologias Utilizadas

Este portfólio foi desenvolvido utilizando as seguintes tecnologias e ferramentas:

- **[Next.js](https://nextjs.org/)** - Framework React para renderização híbrida e estática.
- **[TypeScript](https://www.typescriptlang.org/)** - Superset de JavaScript que adiciona tipagem estática (86% do código).
- **CSS / Tailwind / Styled Components** - Para uma estilização moderna e responsiva.
- **Vercel** - Hospedagem rápida e contínua.

## 🚀 Como rodar o projeto localmente

Se você deseja clonar este repositório para explorar o código, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/DaviMoraisdev/Portfolio2026.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd Portfolio2026
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   # ou yarn install / pnpm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   # ou yarn dev / pnpm dev
   ```

5. **Acesse no seu navegador:**
   Abra [http://localhost:3000](http://localhost:3000) e veja o projeto rodando.
